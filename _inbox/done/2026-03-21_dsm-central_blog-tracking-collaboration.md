### [2026-03-21] Blog Content Tracking Index: New Collaboration Model

**Type:** Action Item
**Priority:** High
**Source:** DSM Central

## What Changed

DSM Central now maintains a living tracking index at `dsm-docs/blog/README.md`
with all 18 blog content files indexed by stage (Seed → Materials → Draft →
Published).

## Your Role

When blog-poster publishes a blog post:

1. Send an inbox entry to DSM Central (`_inbox/`) confirming publication
2. Entry should include: file name published, publication date, blog URL
3. Central will update the README tracking table with the Published date

## Entry Template for Publication Confirmations

```markdown
### [YYYY-MM-DD] Blog published: {short title}

**Type:** Notification
**Priority:** Low
**Source:** dsm-blog-poster

Published `{filename}` on YYYY-MM-DD.
URL: {blog post URL}
```

## Other Changes

- LinkedIn is now a promotion channel only (share link to blog), not a
  separate publication target. The tracking table has no LinkedIn column.
- Blog is the primary publication platform.

## Reference

Full tracking table: `dsm-docs/blog/README.md` in DSM Central.
